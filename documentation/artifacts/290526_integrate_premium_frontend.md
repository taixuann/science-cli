## Objective
Integrate the premium frontend dashboard generated by Google AI Studio into `src/science_cli/serve/frontend` while upgrading the Python web server (`server.py` and `api.py`) to fully support dynamic project discovery and project-level routing context without breaking any existing CLI or data features.

## Context
The AI Studio design enhances our plot gallery with:
- Premium light, dark, and OLED themes with glassmorphic cards and beautiful transitions.
- Interactive multi-project switching dropdown in the sidebar.
- Seamless step navigation and binned statistics.
- Interactive lightbox with panning/zooming.

To make this work dynamically with the canonical server, we must:
1. Update `src/science_cli/serve/server.py` to route `/api/projects` to list active workspaces.
2. Teach the request handler to extract dynamic project overrides from query parameters or `X-Project-Override` headers.
3. Update `src/science_cli/serve/api.py` to resolve absolute paths using `get_projects_root()`.
4. Overwrite `src/science_cli/serve/frontend/` assets with the premium ones without changing directory structure or breaking packaging.

## Specification

### 1. Server-Side APIs & Project Resolution
- **New API Route**: `GET /api/projects`
  - Fetches the configured projects root using `get_projects_root()`.
  - Scans and returns a JSON list of directories representing active projects (e.g. `res_internship`, `non-res_odon-vallet`, etc.).
- **Dynamic Override Extraction**:
  - In `SciServeHandler.do_GET`, extract `project` query param or `X-Project-Override` header.
  - Temporarily assign `self.project_override = project_param` for the scope of the request, allowing on-the-fly switching.
- **Dynamic Path Resolution**:
  - In `api._resolve_project`, if the parameter is a simple directory name (e.g., `res_internship`), resolve it against `get_projects_root()`.

### 2. Frontend Assets Overwrite
- Overwrite `src/science_cli/serve/frontend/index.html` with `documentation/science-cli-plot-gallery/index_standalone.html`.
- Overwrite `src/science_cli/serve/frontend/assets/dashboard.css` with `documentation/science-cli-plot-gallery/assets/dashboard.css`.
- Overwrite `src/science_cli/serve/frontend/assets/dashboard.js` with `documentation/science-cli-plot-gallery/assets/dashboard.js`.
