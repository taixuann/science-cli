## Objective
Establish a unified, publication-quality visual theme and technique template engine across all characterization groups (`memristor`, `ec` [electrochem], `iv`, `raman`, `uv-vis`), incorporating standard margins, DPI levels, tick configurations, legend placements, and technique-specific presets.

## Context
Rendering publication-ready plots (ACS, Nature, Tufte) requires more than just global theme settings like colors and fonts. Each experimental technique requires unique visual parameters: for example, Nyquist plots in Electrochemical Impedance Spectroscopy (EIS) **must** maintain a strict 1:1 square aspect ratio to avoid distortion of impedance semi-circles, while Raman spectra need baseline-correction parameters and peak labeling. The user has generated theme previews previously; this plan codifies these specifications into the codebase themes (`theme/themes/`) and templates (`theme/templates/`).

## Specification

### 1. Symmetrical Themes Configuration (`theme/themes/`)
- Ensure that the publication themes (`publication-acs.yaml`, `publication-nature.yaml`) enforce standard publication parameters:
  - **DPI**: 300 for screen previews, 600 for savefig outputs.
  - **Typography**: Helvetica/sans-serif, size 7-8pt for labels, 6pt for ticks and legends.
  - **Linewidths**: 0.5pt for axes spines and tick marks, 0.75pt to 1.0pt for lines.
  - **Ticks**: Standardize `tick_direction: in` (ticks pointing inside the plot area, which is standard for journals).

### 2. Technique-Specific Presets & Templates (`theme/templates/`)
Create and refine standard templates for each technique group:
- **`raman.yaml`**:
  - Predefine fingerprint region `[200, 1800] cm⁻¹` and high-frequency region `[1800, 3500] cm⁻¹`.
  - Default peak-labeling enabled (`annotate_peaks: true`) with peak prominence set to `15%`.
  - Default baseline correction method: `asls` (Asymmetric Least Squares).
- **`uv-vis.yaml`**:
  - Predefine UV region `[300, 400] nm` and visible region `[400, 800] nm`.
  - Default peak-normalization enabled to scale absorbance curves from 0 to 1.
- **`ec-eis.yaml`**:
  - Enforce `aspect: equal` (1:1 square ratio) for Nyquist plots to ensure accurate geometric representation of impedance semicircles.
  - Default labels: x-axis `Z' (Ω)`, y-axis `-Z'' (Ω)` (negative imaginary impedance).
- **`ec-cv.yaml` & `ec-ca.yaml`**:
  - Enforce standard electrochemistry conventions (e.g. standard IUPAC vs Polarographic potential conventions).
- **`iv-sweep.yaml`**:
  - Default logarithmic y-axis setting (`--loglog` or semilog) enabled for leakage current analysis.

### 3. Dynamic Merging Engine
- Update `src/science_cli/theme/registry.py`:
  - When `sci plot` is called, the engine loads the global theme (e.g., `publication-nature`), loads the technique-specific template (e.g., `ec-eis`), and performs a deep-merge.
  - The technique-specific template overrides the global theme for layout parameters (like aspect ratios and label scales) while preserving the global color palette and font family.
